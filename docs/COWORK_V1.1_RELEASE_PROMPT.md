אתה מוציא את עדכון **v1.1.0** של "מה בתמונה?" לשתי החנויות. כל הקוד כבר בנוי, נבדק ודחוף — אתה מבצע רק את צעדי החנות/הקונסולה. **מקור אמת יחיד: ענף `claude/qa-launch-prep-EXqLn` בריפו `rotem-ya/whoisthere`.** אין לבנות משום ענף אחר.

**סדר העבודה: מתחילים ב-iOS עכשיו. את ה-AAB לאנדרואיד אקבל ואשלח לך בהמשך — אל תחכה לו כדי להתחיל באפל.**

## שלב 1 — iOS (App Store) דרך Codemagic — להתחיל עכשיו
1. הפעל בילד באחת משתי הדרכים:
   - **תג (מועדף):** מהמחשב:
     ```
     git fetch origin claude/qa-launch-prep-EXqLn
     git tag ios-v3 origin/claude/qa-launch-prep-EXqLn
     git push origin ios-v3
     ```
   - **או ידנית:** Codemagic → אפליקציית whoisthere → Start new build → ענף `claude/qa-launch-prep-EXqLn` → workflow `ios-testflight`.
2. Codemagic בונה, חותם ומעלה אוטומטית ל-TestFlight (App Store Connect integration בשם `Apple_Key_Trivia`, Key ID 995PSX889V; bundle `com.rotem.whoisthere`).
3. המתן שהבילד יסיים Processing ב-App Store Connect (~10–30 דק').
4. App Store Connect → מה בתמונה? (Apple ID `6776076758`) → ➕ גרסה חדשה **1.1.0** → בחר את הבילד החדש → הדבק את "מה חדש" (למטה) → Save → **Submit for Review**.
5. אין SDK חדשים — הצהרות הפרטיות נשארות כמו v1.0 (ATT + AdMob כבר מוצהרים).

## שלב 2 — אנדרואיד (Google Play) — כשתקבל ממני את ה-AAB
ה-AAB נבנה כעת ב-CI (ריצה #25 של "Build AAB", חתום במפתח ההעלאה EA:3B, **versionName 1.1.0, versionCode 25**). אשלח לך את הקובץ / קישור ל-artifact כשיסתיים. כשהוא אצלך:
1. אם קיבלת קישור לריצה — ודא שהיא ירוקה ושבלוג שלב האימות מופיע **SHA1: EA:3B** (השלב נכשל אוטומטית אם החתימה שגויה). הורד את ה-artifact `app-release-aab`.
2. Play Console → מה בתמונה? (`com.whoisthere.app`) → המסלול שבו נמצאת v1.0 (Closed testing) → **Create new release** → העלה את ה-AAB → הדבק את הערות הגרסה → Review & Rollout.
3. ⚠️ אם v1.0 (vc22) עדיין בבדיקה ו-Play חוסם רילוס חדש — המתן ותזמן לפי שיקולך.

## נוסח "מה חדש" (לשתי החנויות)
```
• משחק חוזר עם חברים — תוקן ועובד חלק
• מסך ביניים בין תמונות: רואים מי ניחש נכון ומה הייתה התשובה
• משחק שלם בקטגוריה אחת לבחירתכם
• הזמנת חברים בקליק + התראה על בקשות חברות
• שליחת משוב ושיתוף האפליקציה מהפרופיל
• שיפורי הוגנות מול שחקני מחשב ותיקוני יציבות
```

## מה יש בעדכון (רקע, אם צריך תקציר לביקורת)
- תיקון "משחק חוזר" בכל המצבים (מקומות, חי-צומח-דומם, אותיות) + הגנה ממרוץ לחיצות
- סנכרון סבבים: ניחוש נכון סוגר את ההקלדה של כולם + מסך ביניים 4ש׳ עם התמונה, שמה ושם המנחש
- בחירת נושאים: מארח שמכסה את הסבבים פוטר את השאר; משחק בקטגוריה אחת; כפתור ביטול הקלדה
- חברים: קישור הזמנה עובד, שיפור הוספה בקוד, באנר בקשות גלובלי
- v1.1 המקורי: משוב, דיווח קריסות, שיתוף האפליקציה, "האפליקציות שלנו", הגירת אורח→חשבון
- תוכן חי מהאדמין (החלפת תמונות/מקומות חדשים בלי עדכון גרסה)
- בוטים לא מנחשים נכון לפני 50% חשיפת לוח
