# פרומפט ל-cowork — הוצאת עדכון v1.1.0 לשתי החנויות (2026-07-05)

העתק את הטקסט שמתחת לקו לצ'אט של cowork.

---

אתה מוציא את עדכון **v1.1.0** של "מה בתמונה?" לשתי החנויות. כל הקוד כבר בנוי ודחוף — אתה מבצע רק את צעדי החנות/הקונסולה. **מקור אמת יחיד: ענף `claude/qa-launch-prep-EXqLn` בריפו `rotem-ya/whoisthere`, קומיט `a42cf83`.** אין לבנות משום ענף אחר.

## מה יש בעדכון (לתקציר ולהערות הגרסה)
- תיקון "משחק חוזר" (כל המצבים: מקומות, חי-צומח-דומם, אותיות) + הגנה ממרוץ לחיצות
- סנכרון סבבים בחי-צומח-דומם: ניחוש נכון סוגר הקלדות של כולם + מסך ביניים של 4 שניות עם התמונה, שמה ושם המנחש
- בחירת נושאים: מארח שמכסה את כל הסבבים פוטר את השאר; משחק בקטגוריה אחת; כפתור ביטול הקלדה
- חברים: קישור הזמנה עובד, שיפור הוספה בקוד, באנר בקשות גלובלי מכל מסך
- v1.1 המקורי: משוב, דיווח קריסות אוטומטי, שיתוף האפליקציה, "האפליקציות שלנו", הגירת אורח→חשבון
- תוכן חי מהאדמין (החלפת תמונות/מקומות חדשים בלי עדכון גרסה)
- בוטים לא מנחשים נכון לפני 50% חשיפה

### נוסח מוצע ל"מה חדש" (עברית)
```
• משחק חוזר עם חברים — תוקן ועובד חלק
• מסך ביניים בין תמונות: רואים מי ניחש נכון ומה הייתה התשובה
• משחק שלם בקטגוריה אחת לבחירתכם
• הזמנת חברים בקליק + התראה על בקשות חברות
• שליחת משוב ושיתוף האפליקציה מהפרופיל
• שיפורי הוגנות מול שחקני מחשב ותיקוני יציבות
```

## אנדרואיד (Google Play) — עם ה-AAB שכבר נבנה
1. פתח: https://github.com/rotem-ya/WhoIsThere/actions/workflows/build-aab.yml — ריצה **#25** (ענף `claude/qa-launch-prep-EXqLn`, קומיט `a42cf83`).
2. ודא שהריצה ירוקה ושבלוג שלב האימות מופיע **SHA1: EA:3B** (השלב נכשל אוטומטית אם החתימה לא נכונה — אם נכשל, עצור ודווח).
3. הורד את ה-artifact בשם `app-release-aab` (קובץ `app-release.aab`). **versionName 1.1.0, versionCode 25** (אוטומטי ממספר הריצה — גדול מ-23 של ההשקה).
4. Play Console → מה בתמונה? (`com.whoisthere.app`) → אותו מסלול שבו נמצאת v1.0 (Closed testing) → Create new release → העלה את ה-AAB → הדבק את הערות הגרסה → Review & Rollout.
5. ⚠️ v1.0 (vc22) אולי עדיין בבדיקה — אם Play לא מאפשר רילוס חדש עד סיום הבדיקה, המתן ותזמן לפי שיקולך.

## iOS (App Store) — דרך Codemagic
1. הפעל בילד באחת משתי הדרכים:
   - **תג (מועדף):** מהמחשב: `git fetch origin claude/qa-launch-prep-EXqLn && git tag ios-v3 origin/claude/qa-launch-prep-EXqLn && git push origin ios-v3`
   - **או ידנית:** Codemagic → אפליקציית whoisthere → Start new build → ענף `claude/qa-launch-prep-EXqLn` → workflow `ios-testflight`.
2. Codemagic בונה, חותם ומעלה אוטומטית ל-TestFlight (integration בשם `Apple_Key_Trivia`; build name 1.0.0 בקונפיג — אם רוצים 1.1.0 מוצג למשתמשים, ודא ש-`codemagic.yaml` בענף כבר קובע 1.1.0; זה כבר עודכן בקוד).
3. המתן שהבילד יסיים Processing ב-App Store Connect (~10–30 דק').
4. App Store Connect → מה בתמונה? (Apple ID `6776076758`) → ➕ גרסה חדשה **1.1.0** → בחר את הבילד החדש → הדבק "מה חדש" → Save → Submit for Review.
5. ודא שאין שינוי בהצהרות פרטיות (אין SDK חדשים — אותן הצהרות כמו v1.0).
