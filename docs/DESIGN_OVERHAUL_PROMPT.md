# פרומפט לקלוד עיצוב — שדרוג ויזואלי וסאונד מקיף ל-WhoIsThere

## מה זו האפליקציה
WhoIsThere ("מה בתמונה?") היא אפליקציית מובייל (Flutter, iOS + Android) עם כמה משחקי ניחוש מולטיפלייר בעברית (RTL): זיהוי תמונה מתוך משבצות שנחשפות בהדרגה, חי-צומח-דומם, "זהו את הפתגם", ומשחק אותיות בסגנון וורדל. יש חדרים משותפים (חברים/אקראי), חנות קוסמטיקה (סקינים, מסגרות אווטר, אפקטי ניצחון), כלכלת מטבעות, מערכת חברים וקבוצות. היא כבר חיה בשתי החנויות (App Store + Google Play) עם משתמשים אמיתיים.

**זהות ויזואלית נוכחית:** רקע כחול-לילה עמוק (`#07101F`) עם גוונים (`#0E1E35`), מבטא זהב (`#D4AF37`), הבהוב ציאן (`#00F2FF`/`#87CEEB`) לאלמנטים אינטראקטיביים, פונט Hebrew/Heebo, כרטיסים מוארים בעדינות עם shadow/glow. יש כבר: אנימציית קונפטי בסיסית בניצחון, סאונד ניצחון (fanfare), אפקטי "חלקיקים" (win effects) בחנות, מסגרות אווטר עם glow, ואפקט "vault" (כספת) לפתיחת משבצות.

## המטרה
להעלות את רמת העיצוב והסאונד ב-100 דרגות: מרמה של אפליקציה עצמאית טובה, לרמה של אפליקציה מקצועית ברמת ראש רשימת ה-Top Charts (Duolingo, Candy Crush, Clash Royale, Monument Valley, Trivia Crack). **התקציב פתוח** - מותר ורצוי להמליץ על רכישת אסטים מקצועיים במקום לבנות הכל מאפס.

**הצעת כיוון תקציבית (אפשר לחלוק עליה):** נכסי UI כלליים (קליקים, אנימציות מעברים) - חבילות מוכנות זולות/חינמיות (למשל Kenney.nl, LottieFiles). עבודה מותאמת אישית בתשלום (Fiverr/Upwork) שמורה לרגע אחד שהכי משתלם להשקיע בו: מסך הניצחון.

## דרישה מרכזית: מיפוי אירוע-לאירוע, לא המלצות כלליות
**לכל אירוע ברשימה למטה** (טבלה בסעיף הבא) אני צריך שורה מלאה: מה קורה היום (שקט/אנימציה בסיסית) → מה מומלץ (סאונד ספציפי + אנימציה ספציפית) → **קישור הורדה ישיר** לנכס המומלץ (או שם מוצר מדויק + שוק + מונח חיפוש מדויק, אם אין לך גישה לדפדף בפועל) → הערת מימוש ב-Flutter (איזה package, בערך כמה שורות קוד). בלי הרשימה הזו הפלט לא שמיש.

## מוצרי חנות - עיצוב מחדש (לא רק UI סביבם, הנכסים עצמם)

יש חמש קטגוריות קוסמטיקה. חלקן מיוצרות היום דרך Gemini במסך אדמין ייעודי (יש שם pipeline קיים ליצירה+פרסום), חלקן קבצי asset סטטיים. לכל קטגוריה: תגיד אם הכי משתלם לשפר את **פרומפט הג'נרציה** (Gemini) או **לקנות/להזמין** נכסים במקום, ולמה.

1. **גב כרטיס / card skins** (16 עיצובים, 640×640, מדרגות מחיר 0/50/100/200/500/1000): מינימליסטי, טבע ישראלי, פסיפס מזרחי, ניאון, קוסמי, זהב מלכותי יחיד. אלה התמונות שמופיעות כרקע המשבצות בלוח המשחק - הכי נראות, הכי משתלם לשדרג.
2. **מסגרות אווטר / avatar frames** (13, none עד legend): טבעת גרדיאנט סביב תמונת השחקן, כרגע קוד (SweepGradient), לא תמונה.
3. **צבעי שם / name styles** (14): אפקט צבע/גרדיאנט על טקסט שם השחקן.
4. **רקעי לוח / board skins** (13, none עד emerald_dream): רקע מסך המשחק כולו מאחורי המשבצות.
5. **אווטרים / avatar choices** (54 פריטים, אימוג'י + תמונת Gemini פר-פריט כשמפורסם): התשתית כבר קיימת (מסך אדמין עם יצירת Gemini פר-פריט/batch, פרומפט "פורטרט" לפי מדרגת מחיר) - כאן הכי הגיוני לשפר את **הפרומפט** לג'נרציה ולא לקנות נכסים חיצוניים, כי כבר יש pipeline.

## מה עוד מבוקש (מעבר למיפוי האירועים)

### "מסיבת ניצחון" (win celebration) - הרחבה
רצף חגיגה רב-שכבתי, בהשראת פתיחת תיבה ב-Clash Royale או ניצחון ב-Duolingo: בילד-אפ קצר (מתח) → פיצוץ ויזואלי (קונפטי + חלקיקים + מסגרת זהב מאירה + זום על שם המנצח) → haptic מתוזמן לשיא → אם יש תגמול מטבעות, ספירה אנימטיבית של המספר.

### רמת גימור כללית
עקביות מרווחים/פונטים/פינות/צללים לפי מערכת עיצוב אחידה (design tokens), לא מספרים שרירותיים בכל מסך. לשקול אייקונוגרפיה מקצועית אחידה במקום אימוג'ים בחלק מהמקומות.

## אילוצים לשמור עליהם
- **RTL עברית** בכל מקום
- **ביצועים במובייל** - להעדיף אנימציות Flutter קלות (AnimationController, Rive קל) על פני Lottie כבד בכל מקום
- **זהות הצבע הקיימת** (כחול-לילה + זהב) מוכרת כבר למשתמשים - להעמיק/לעדן, לא בהכרח להפוך לגמרי
- **גודל האפליקציה** - אסטים כבדים דורשים שיקול דעת, האפליקציה חיה בחנויות עם מגבלות הורדה

## מה אני צריך כפלט (בסדר הזה)
1. **הטבלה המלאה** (אירוע → קיים היום → מומלץ סאונד → מומלץ אנימציה → קישור/שם מוצר מדויק → הערת Flutter)
2. **טבלת מוצרי חנות** (קטגוריה → פרומפט Gemini משופר / המלצת רכישה → קישור אם רלוונטי)
3. **מפרט מסך הניצחון המלא** (הרצף המורחב, כולל timing בשניות לכל שלב)
4. **תוכנית שלבים** - האפליקציה חיה עם משתמשים, אז מדורג: מה נכנס ראשון (סיכון נמוך, אימפקט גבוה), מה יכול לחכות
