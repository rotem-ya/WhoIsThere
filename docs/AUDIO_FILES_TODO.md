# קבצי אודיו להשלמת בוסט הסאונד (v1.4.3)

כל ה‑hooks כבר בקוד ומתנגנים **ברגע** שמכניסים את הקובץ בשם ובנתיב המדויקים. עד אז
הכל שקט (fail‑soft). פורמט מומלץ: SFX ב‑`.ogg` (קטן), מוזיקה ב‑`.mp3`. שמור עוצמות
מנורמלות (‑3dB peak) כדי שלא יהיו קפיצות עוצמה.

## אפקטי סאונד → `assets/sounds/ui/`
| קובץ | רגע במשחק | אורך | אופי |
|------|-----------|------|------|
| `transition.ogg` | מעבר בין סבבים (מסך ביניים) | ~0.4ש' | וווש קצר, מעופף |
| `streak.ogg` | רצף פגיעות במשחק האותיות | ~0.3ש' | פינג עולה, מתגבר |
| `tile_flip.ogg` | חשיפת משבצת (flip) | ~0.2ש' | היפוך קלף רך |
| `coin_shower.ogg` | מטבעות עפים לארנק | ~0.8ש' | מפל מטבעות עשיר |
| `spin_tick.ogg` | טיק בזמן סיבוב הגלגל | ~0.08ש' | טיק ראצ'ט בודד, חד |
| `spin_land.ogg` | נחיתת גלגל המזל על פרס | ~0.3ש' | "טאנק" מספק |
| `quest_complete.ogg` | איסוף משימה יומית | ~0.7ש' | ג'ינגל עליז קצר |
| `heartbeat.ogg` | 3 השניות האחרונות בספירה | ~0.3ש' | פעימת לב בודדת, עמומה |

> יש כבר קבצי `ui/*.ogg` קיימים (click/coin_gain/rank_up וכו') — לא לגעת בהם.

## מוזיקת רקע → `assets/sounds/`
נגן יחיד, לופ. **חייב לופ חלק (seamless)** בלי חתך בתחילה/סוף.
| קובץ | מתי מתנגן | אורך | אופי |
|------|-----------|------|------|
| `music_menu.mp3` | בית / חברים / חנות / פרופיל / הגדרות / טבלה | 30‑60ש' לופ | רגוע‑אנרגטי, ידידותי |
| `music_lobby.mp3` | לובי / חיפוש יריב | 20‑40ש' לופ | קליל, ממתין |
| `music_win.mp3` | מסך ניצחון (ראוט /win) | 10‑20ש' לופ | חגיגי קצר |

- העוצמה מנוהלת אוטומטית לפי סליידר המוזיקה בהגדרות (מוכפל ב‑0.5 שיישב מתחת ל‑SFX).
- מוזיקת התפריט **נעצרת אוטומטית** בתוך המשחק (game/letters/vote) — רק נגן המשחק פועל שם, כדי למנוע שני זרמי mp3 בו‑זמנית (קריסת iOS ידועה). לכן אין צורך לגעת בקוד.

## נדחה בכוונה (לא בבוסט הזה)
- **מוזיקה פר‑מצב משחק** ו**מוזיקת מתח באנדגיים** — נוגעות בנגן ה‑bg הפעיל של המשחק
  (`background_studio.mp3` שעובד). כדי לא להשתיק את המוזיקה במשחק אם קובץ חסר, צריך
  loader מודע‑קיום. אפשר לעשות בסבב נפרד אחרי שמפיקים את הקבצים.

## איך בודקים שעובד
מכניסים קובץ אחד → בונים → מפעילים את הרגע המתאים באפליקציה. אם שקט: לוודא שם/נתיב
מדויקים ושהסליידר לא ב‑0.
