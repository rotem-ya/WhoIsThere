# כיווני עיצוב — WhoIsThere

מוקאפים של כיווני עיצוב לשדרוג ה"רמה" של האפליקציה. פותחים כל HTML בדפדפן.
אחרי בחירת כיוון — מממשים אותו כקו עיצובי אחיד בכל המסכים, עם סאונד מתאים לכל פעילות.

## עקרונות משותפים (יחולו על הכיוון הנבחר)
- קו עיצובי **אחד** בכל המסכים: אותה פלטה, טיפוגרפיה, מרווחים, רדיוסים, כפתורים.
- **סאונד מתאים לכל פעילות** — התשתית כבר בקוד (`SfxService`); כל אירוע מקבל צליל הולם.
- מוטיב חתימה חוזר (ב-01: הצמצם/aperture מ-VaultCover).
- אנימציה מבוססת-אסט לרגעים גדולים (Lottie/Rive) — המנוע כבר בקוד (ראה `game_winner_view.dart`).

## מקורות אסטים (לכיוון הנבחר)
Lottie/Rive מותאם (Fiverr/LottieFiles/rive.app) · חבילות (Craftpix/Envato/UI8) · תלת-ממד (Spline) · או מאייר יחיד לסט אייקונים.
