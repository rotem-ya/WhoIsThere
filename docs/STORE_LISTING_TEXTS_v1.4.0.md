# טקסטים לחנויות — גרסה 1.4.2 (Candy + פוליש)

> **הערה:** גרסת Candy כבר חיה בגוגל (1.4.1). 1.4.2 היא עדכון פוליש מעליה:
> אנימציית מטבעות עפים בזכיות, לובי חלק יותר עם כניסת שחקנים, מעברי מסך רכים,
> וחגיגת "פריט פרימיום נפתח" בחנות. התיאור הראשי נשאר זהה; "מה חדש" עודכן לפוליש.

כל הטקסטים בעברית, מוכנים להעתקה. בלי מקף ארוך, בסגנון אנושי (לפי `docs/NO_AI_TELLS_STYLE.md`).
שם האפליקציה: להשאיר את הקיים (**מה בתמונה?**). לא לשנות שם בהגשת עדכון.

## Google Play — דף החנות הראשי

### תיאור קצר (עד 80 תווים)
```
חשפו את התמונה, נחשו מה מסתתר בה לפני כולם, ושחקו עם חברים.
```

### תיאור מלא (עד 4000 תווים)
```
מה בתמונה? משחק ניחוש תמונות מהיר וממכר.

בכל סבב נחשפת תמונה מוסתרת משבצת אחר משבצת. המטרה פשוטה: לזהות מה בתמונה לפני כל השאר. ככל שתנחשו מוקדם יותר, תרוויחו יותר מטבעות.

מצבי משחק:
• זיהוי מקומות: מצאו את המקום שבתמונה.
• חי צומח דומם: חיות, צמחים, חפצים ועוד, בסבבים מהירים.
• זהו את הפתגם: תמונה שרומזת על פתגם עברי מוכר.
• משחק האותיות: נחשו את המילה הנסתרת אות אחר אות.

משחק עם חברים:
הזמינו חברים לחדר פרטי, הקימו קבוצה קבועה, והתחרו על טבלת הניקוד. הכל בזמן אמת, עם צ'אט ותחבולות שמערבבות את היריבים.

התאמה אישית:
פתחו רקעים חדשים ללוח וגב כרטיסיות בעשרות גוונים. הסקינים היקרים נוצצים ומונפשים, כדי שכל לוח ייראה משלכם.

התקדמות:
אספו מטבעות, עלו בדרגות מסקרן ועד אגדה, וגלו עוד ועוד תמונות.

הורידו עכשיו, פתחו את המשחק, ותגלו כמה מהר אתם באמת.
```

### מה חדש (Release notes, עד 500 תווים)
```
ליטשנו את החוויה! מטבעות עפים אליכם בכל זכייה, הלובי חלק יותר עם כניסת שחקנים מונפשת, ומעברים רכים בין המסכים. הוספנו חגיגה מיוחדת כשקונים פריט פרימיום בחנות, ושיפרנו סאונד ואנימציות בכל המשחק. נחשו מה בתמונה, שחקו עם חברים, ואספו מטבעות.
```
